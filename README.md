# Explitter Program

The only program that you will need ever for storing your expenses!

## Prerequisites

1. MariaDB
2. Python
3. Pip

## Setup

1. Run setup sql on root db

```bash
sudo mysql
source setup.sql
exit
```

2. Install the `mysql-connector-python` library

```bash
pip install mysql-connector-python
```

4. Start the program

```bash
python main.py
```

3. Enjoy!
